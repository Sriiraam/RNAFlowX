# ADR-010 Dataset Freeze

## Status

Accepted

## Decision

RNAFlowX v1.0 will use the PBS versus Gluconate
comparison from GSE330905.

## Dataset

GSE330905

## BioProject

PRJNA1465899

## SRA Study

8

## Rationale

Balanced design.

Publicly available.

Human paired-end bulk RNA-seq.

Suitable for demonstrating a complete
RNA-seq differential expression workflow.

## Consequences

Dataset will remain frozen for RNAFlowX v1.0.
