# ADR-009 Data Acquisition

## Status

Accepted

## Decision

RNAFlowX will acquire all input data from official public sources.

## Dataset Source

GEO: GSE330905

## Raw Data Source

SRA: SRP700373

## Reference Source

GENCODE Release 50

## Rationale

Official sources ensure:

- Reproducibility
- Traceability
- Data integrity

## Consequences

Downloaded data must be documented and version tracked.
