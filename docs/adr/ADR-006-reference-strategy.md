# ADR-006 Reference Strategy

## Status

Accepted

## Decision

RNAFlowX will use GENCODE human reference data
based on the GRCh38 assembly.

## Genome

GRCh38

## Annotation Source

GENCODE

## Rationale

GENCODE provides a comprehensive and
widely adopted reference annotation for
human transcriptomic analyses.

## Consequences

All workflow validation and testing
will use GENCODE-based references.
