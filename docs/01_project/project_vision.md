# RNAFlowX

Production-grade RNA-seq workflow platform built with Nextflow DSL2.

## Goal

Provide a reproducible, scalable and maintainable RNA-seq workflow for transcriptomics analysis.

## Target Users

- Bioinformaticians
- Computational Biologists
- Research Scientists

## Key Features

- Quality Control
- Alignment
- Quantification
- Differential Expression Analysis
- Reporting
